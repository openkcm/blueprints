# CMK 

OpenKCM CMK

## Plugins

In order for CMK application/service to run does require to have in place the following plugins:

- Keystore 
- SystemInformationService (sis)
  - [External plugin as binaries](./plugins/sis)
  - [BuiltIn plugin](./plugins/sis-builtin)
- Certificate Issuer
- Notification
